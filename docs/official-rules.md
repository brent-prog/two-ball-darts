# TWO BALL DARTS Official Rules

## Basic game

- 18 holes
- Hole numbers 1-18 are the target numbers
- All holes are par 3
- Each player throws exactly two darts per hole
- Lowest score wins

## Scoring

| Result | Relative score | Rule |
|---|---:|---|
| Eagle | -2 | Both darts hit DOUBLE or TRIPLE of the target number. |
| Birdie | -1 | One dart hits DOUBLE/TRIPLE target + one SINGLE target hit. |
| Par | 0 | Two SINGLE target hits, OR one DOUBLE/TRIPLE target + one on-board miss. |
| Bogey | +1 | One SINGLE target hit + one on-board miss, OR one DOUBLE/TRIPLE target + one off-board dart. |
| Double Bogey | +2 | No target hits with both darts on-board, OR one SINGLE target hit + one off-board dart. |
| Triple Bogey | +3 | No target hits + at least one off-board dart. |

## Local rulings

- Only the active hole number counts as a target hit. Everything else is a miss.
- Bulls count for nothing - ever.
- A bull is still an on-board miss.
- Off the board literally means off the board.
- If a dart sticks anywhere in the cork/board, it is on-board and OK.
- Worst possible score on any hole is triple bogey.

## Tie-break

After 18 holes, if players are tied, each tied player throws one dart. Closest to the bullseye wins.
